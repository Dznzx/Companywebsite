Tech stack

Built production-grade
from day one.

⚛️
React.js

Frontend SPA with Recharts for data visualisation, Tailwind CSS for styling, React Router for navigation.

⚡
FastAPI

Python backend with REST API endpoints for students, auth, CSV upload, and ML predictions. Auto-generates Swagger docs.

🐘
PostgreSQL

Neon Cloud serverless Postgres database. Stores users and student records with SQLAlchemy ORM.

🤖
Scikit-learn

3 ML models trained, evaluated, and serialised with joblib. Logistic Regression, Decision Tree, and Random Forest.

🔐
JWT Auth

Secure token-based authentication with bcrypt password hashing and protected API routes.

▲
Vercel

Frontend deployed on Vercel with automatic GitHub CI/CD. Zero-config deployment.

🖥️
Render

FastAPI backend hosted on Render with environment variable management and auto-deploy on push.

📊
Recharts

Bar charts, pie charts, and distribution graphs for CGPA and risk analytics — all interactive.
